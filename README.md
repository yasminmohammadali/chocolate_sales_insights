<h1>🍫 Chocolate Sales Dashboard (Tableau)</h1>

<h2>📌 Description</h2>
<p>
This project presents an <b>interactive Tableau dashboard</b> built to analyze chocolate sales data across 
<b>brands, countries, pricing, and sales channels</b>.
</p>
<p>
The dashboard provides a comprehensive view of <b>revenue performance, price distribution, product segmentation, and monthly sales trends</b>, 
helping uncover key business insights and support data-driven decision-making.
</p>

<h2>🚀 Key Features</h2>

<h3>🔹 Revenue & KPI Analysis</h3>
<ul>
<li>Total Revenue: <b>$716,696</b></li>
<li>Average Price: <b>$13.78</b></li>
<li>Average Units Sold: <b>104.9</b></li>
<li>High-level performance indicators for quick insights</li>
</ul>

<h3>🔹 Pricing Insights</h3>
<ul>
<li>Price range distribution using histogram</li>
<li>Country-wise price comparison</li>
<li>Identification of pricing variations across markets</li>
</ul>

<h3>🔹 Brand Performance</h3>
<ul>
<li>Units sold by major brands (Cadbury, Ferrero, Hershey, Lindt, Mars, Nestlé, Toblerone)</li>
<li>Monthly sales trends by brand</li>
<li>Comparative analysis of brand performance</li>
</ul>

<h3>🔹 Product Analysis</h3>
<ul>
<li>Breakdown of chocolate types (Milk, Dark, Bars, Boxes)</li>
<li>Contribution of each product category to overall sales</li>
</ul>

<h3>🔹 Sales Channel Analysis</h3>
<ul>
<li>Performance across Convenience Stores, Online, Supermarkets, and Specialty Stores</li>
<li>Channel-wise revenue contribution</li>
</ul>

<h3>🔹 Geographic Insights</h3>
<ul>
<li>Country-wise pricing analysis</li>
<li>Revenue distribution across global markets</li>
</ul>

<h3>🔹 Interactive Filters</h3>
<ul>
<li>Brand selection</li>
<li>Product type filtering</li>
<li>Sales channel filtering</li>
<li>Dynamic interaction across all dashboard visuals</li>
</ul>
